This app created for code example  
And idea, for this chat was inspired by [this](https://habr.com/post/127525/) ancient habr article 


Live demo available [here](https://cooing-song.surge.sh/).

Front deployed with <http://surge.sh/>  
Server deployed with <https://zeit.co/now>

App based on _create-react-app_ so, as package manager it uses _yarn_  
Contain basic tests for components, just for example only

Basically use:  
**yarn install**  
**yarn start**  
**yarn test**  
etc.
